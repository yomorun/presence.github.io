# Backend for Presencejs

The open source backend for Presencejs v2.0

## Features

- WebTransport support
- Websocket arraybuffer mode support
- Zero-copy upgrade to WebSocket
- `SO_REUSEPORT` on Darwin and Linux
- Native Ping/Pong frame implemented in WebSocket
- Support WebTransport Datagrams, unreliable and fast communication
- Geo-distributed Clusters by YoMo

## Working-in-Process

- Reuse goroutine
- WebTransport Stream support, reliable
- pprof
